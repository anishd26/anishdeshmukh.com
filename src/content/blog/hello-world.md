This is the first post on my new site. I built it with [Astro](https://astro.build)
and it's hosted for free on GitHub Pages.

To write a new post, just add another Markdown file to `src/content/blog/`. Each file
needs a small block at the top (the "frontmatter") with a title and date:

```md
---
title: "My second post"
description: "A short summary."
pubDate: 2026-06-24
tags: ["example"]
---

Write your post here using Markdown.
```

That's it — the new post shows up on the blog automatically.
